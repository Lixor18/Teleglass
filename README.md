# Teleglass
Future Telegram with Liquid Glass and free posibilities

How to use?
IMPORTANT!
1. install Python 3.13+
2. paste in terminal/cmd: pip install fastapi uvicorn telethon starlette

How to use the app?
1. Download the lattest release
2. Log in https://my.telegram.org/auth?to=apps
3. Create new app
4. Copy ID and HASH, and paste the password in Teleglass.py in line 19, 20
5. Launch Teleglass.py
